Now Time Mod 准点报时模组

About This Mod 模组介绍

Now Time is a server-friendly Minecraft mod that sends automated time announcements to in-game chat. Highly customizable via config files, it supports multilingual announcements and flexible scheduling.

Now Time 是一款服务端友好的 Minecraft 模组，能在游戏聊天区自动发送准点报时。 通过配置文件可实现高度自定义，支持多语言报时和灵活的时间调度。

Key Features 核心功能

Accurate Alerts

精准报时

Automatic in-game announcements at full hours (e.g. "It's now 12:00")

整点自动发送游戏内公告（例："现在时间是12:00"）

Full Customization

深度定制

Customize message format, trigger times, and language via YAML config

通过YAML配置自定义消息格式、触发时间和语言

Server-Optimized

服务端支持

Runs on dedicated servers without client-side installation

可在专用服务器运行，无需客户端安装

Configuration 配置选项

Parameter 参数 Description 说明 Default 默认值
enable_mod 启用模组 Toggle mod on/off 模组开关 true true
announce_hours 报时时段 Which hours to announce (0-23) 需要报时的时间段(0-23) [9,12,15,18,21] [9,12,15,18,21]
message_format 消息格式 Custom message with {time} placeholder 自定义消息格式，{time}为占位符 "It's now {time}" "现在时间 {time}"
use_24h_format 24小时制 Use 24-hour time format 是否使用24小时制 false false
